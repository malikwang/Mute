# Mute
&emsp;&emsp;这是一个Mac上的小程序，主要实现拔掉耳机时，使电脑静音，避免一些场景下的尴尬情景。
&emsp;&emsp;同时也特意添加对网易云音乐的支持，实现拔掉耳机时，暂停播放。
